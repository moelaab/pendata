Di halaman ini, Anda akan menginstal dan menjalankan WAHA, mengautentikasi klien menggunakan kode QR, dan mengirim pesan pertama Anda ke WhatsApp menggunakan API!

Kami akan memandu Anda melalui langkah-langkah yang diperlukan untuk berhasil mengirim pesan teks pertama Anda menggunakan API WhatsApp.

7.1 Apakah Anda pembelajar visual?

Kami punya solusinya! Tonton video tutorial di bawah untuk melihat cara mengirim pesan pertama Anda menggunakan WAHA

7.2 Langkah 0. Persyaratan

WAHA berjalan di atas Docker, hanya itu yang Anda butuhkan!

👉 Silakan ikuti panduan resmi Docker untuk menginstalnya di Linux, Windows, dan macOS

Mengapa Docker?

7.3 Langkah 1. Unduh image

Dengan asumsi Anda telah menginstal Docker, mari unduh image-nya! Ikuti instruksi di bawah ini:

WAHA Core | WAHA Plus | WAHA (ARM)

Jalankan perintah di bawah ini:

docker pull devlikeapro/waha

7.4 Langkah 2. Inisialisasi WAHA

Buat file .env yang akan kita gunakan di langkah berikutnya untuk kredensial:

Init WAHA

docker run --rm -v "${PWD}:/app/env" devlikeapro/waha init-waha /app/env

Init WAHA (Output)

Generated env values:
  - WAHA_API_KEY=8d18f1dee88945cd9aa0c1a324545f8d
  - WAHA_API_KEY_PLAIN=8d18f1dee88945cd9aa0c1a324545f8d
  - WAHA_DASHBOARD_USERNAME=admin
  - WAHA_DASHBOARD_PASSWORD=0ed3b6c5655e4685b16adccb29519250
  - WHATSAPP_SWAGGER_USERNAME=admin
  - WHATSAPP_SWAGGER_PASSWORD=0ed3b6c5655e4685b16adccb29519250

Use these credentials to login in Dashboard or Swagger:
  - Username: admin
  - Password: 0ed3b6c5655e4685b16adccb29519250

Use this API key in the x-api-key header:
  - 8d18f1dee88945cd9aa0c1a324545f8d

Read more:
  - https://waha.devlike.pro/docs/how-to/dashboard/#api-key
  - https://waha.devlike.pro/docs/how-to/security/

Ingat nilai-nilai ini (Anda selalu dapat memeriksa file .env jika Anda lupa):

  • Username / Password: admin / 11..11 - gunakan untuk mengakses Dashboard dan Swagger UI
  • Api Key: 00..00 - gunakan untuk terhubung ke server Anda

💡 Anda dapat mengubah variabel ke nilai apa pun, tetapi gunakan string panjang (seperti UUIDv4)

7.5 Langkah 3. Jalankan WAHA

Jalankan WhatsApp HTTP API:

docker run -it --env-file "$(pwd)/.env" -v "$(pwd)/sessions:/app/.sessions" --rm -p 3000:3000 --name waha devlikeapro/waha

7.8 Langkah 6. Kirim pesan teks

Sekarang kita siap mengirim pesan pertama ke WhatsApp melalui API!

  • Ganti 123123 dengan nomor telepon Anda tanpa +, tetapi tetap pertahankan bagian @c.us.
  • Gunakan API key dari file .env Anda (WAHA_API_KEY).

Contoh di bawah menggunakan placeholder 00000000000000000000000000000000 - tukar dengan nilai asli Anda:

curl | JavaScript | Python

curl -X 'POST' \
  'http://localhost:3000/api/sendText' \
  -H 'Accept: application/json' \
  -H 'Content-Type: application/json' \
  -H 'X-Api-Key: 00000000000000000000000000000000' \
  -d '{
  "chatId": "123123@c.us",
  "text": "Hi there!",
  "session": "default"
}'

Sebagai alternatif untuk curl, Anda dapat menggunakan 🦜 Swagger untuk itu.

Buka Swagger di http://localhost:3000/#/chatting.

Authorize menggunakan Api Key (file .env - WAHA_API_KEY):

  • Gulir ke bawah ke bagian chatting.
  • Temukan endpoint POST /api/sendText dan expand
  • Klik Try it out
  • Ganti 123123@c.us dengan nomor telepon Anda tanpa +, tetapi tetap pertahankan bagian @c.us
  • Klik Execute
